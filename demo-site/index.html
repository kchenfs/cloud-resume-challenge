This is a condensed resume website built using various AWS services. I wrote my resume in HTML and styled it with CSS. Then, I deployed my resume as an S3 static website and enabled HTTPS using CloudFront for security. I used Route 53 as my DNS provider to point a custom domain name to the CloudFront distribution. Additionally, I included a visitor counter on my website using JavaScript and stored and updated the visitor count in DynamoDB. I created an API using AWS API Gateway and Lambda to interact with DynamoDB. For writing the Lambda function, I used Python and the boto3 library for AWS. I also included tests for my Python code. I defined my API resources in an AWS Serverless Application Model (SAM) template and deployed the SAM application using the AWS SAM CLI for infrastructure as code. Finally, I set up separate GitHub repositories for my backend and website code, and used GitHub Actions for continuous integration and deployment of both the backend and frontend code.
